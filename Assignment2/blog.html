All three of them are used to declare variables. In case of const, once a variable is declared with const, the value of the variable can not be changed after. With var and let, the change of values is possible. let keyword binds the variable inside a block meaning that variable can't be accessed outside the block if that is declared inside a block. In other words, the variable won't be global. Finally, with the var keyword, global variables will be declared meaning the variable can be accessed outside the blocks even if the variable is declared in a block.

Null is basically what we programmers assign in a variable that is empty. While undefined is given by javascript when we don't assign any value to a variable. A function returns undefined when nothing is returned. In short, null is intentional absense of value and undefined is somewhat a default value when no value is assigned.

Full form of REST API is- Representational State Transfer API. This is an architectural style for designing networked applications. It's used to build a communication between client and server in a standardized and efficient way over the web.
